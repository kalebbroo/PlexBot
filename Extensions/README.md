# PlexBot Extensions

Place your extension folders here. Each extension should be its own subdirectory containing a `.csproj` file and source code.

Extensions are automatically built from source and loaded when PlexBot starts. To disable an extension without removing it, rename its folder with a `.disabled` suffix (e.g., `MyExtension.disabled`).

See [Creating Extensions](../Docs/Extensions/CreatingExtensions.md) for full documentation.
